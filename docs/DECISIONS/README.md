# Architectural Decision Records

This directory contains ADRs — short documents that record significant architectural decisions, their context, and their consequences.

## Format

Each ADR is a markdown file numbered sequentially: `0001-decision-title.md`.

## Status values

- **Proposed** — under discussion
- **Accepted** — current
- **Superseded** — replaced by a later ADR (link to it)
- **Deprecated** — no longer relevant

## Template

```markdown
# ADR-XXXX: <Title>

**Status:** Accepted
**Date:** YYYY-MM-DD
**Author:** <name>

## Context

What problem are we solving? What constraints apply?

## Decision

What did we decide?

## Consequences

Positive, negative, neutral.

## Alternatives considered

What else did we look at and why did we reject it?
```
